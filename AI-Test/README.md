# Healthcare LNP AI Model

A Language Neural Processing (LNP) model for healthcare applications, focusing on disease prediction, medication suggestion, and patient risk assessment.

## Features

- **Disease Classification**: Predicts disease categories from medical text
- **Medication Suggestions**: Recommends appropriate medications
- **Severity Assessment**: Evaluates condition severity (Mild/Moderate/Severe)
- **Risk Scoring**: Calculates patient risk scores
- **Symptom Extraction**: Identifies key symptoms from medical notes
- **Web Interface**: User-friendly Streamlit app for medical professionals
